Bathymetry Data Analysis and Processing

Bathymetry survey bring new challenges to the process of obtaining and processing the data. Water is completely different environment and the subsequent data processing is crucial for creating accurate and useful bathymetric maps.

Data Cleaning

  • Remove outliers and errors in the collected data.
  • Address issues like signal noise, interference, or artifacts.

Georeferencing

  • Align depth data with accurate geographic coordinates.
  • Use GPS and inertial navigation system data for precise positioning.

Grid Generation

  • Create a regular grid from the point data for easier analysis.
  • Interpolate data between survey lines for a continuous representation.

Bathymetric Model Creation

  • Use interpolation methods (e.g., kriging, TIN interpolation) to create a 3D bathymetric model.
  • Combine depth information with backscatter data for a more detailed representation.

Map Production

  • Generate bathymetric maps using GIS (Geographic Information System) software.
  • Apply color gradients or contour lines to represent depth variations.

Sonar Mosaic

  • Sonar mosaic is way of interpreting sonar images into one large mosaic which describes a phenomenon in its entirety. This work process was developed in collaboration with the company AUSROV.

Report Generation

  • Compile the results into a comprehensive report, including methodology, data quality assessments, and interpretations.
